## Governance & Decision-Making

**Project Governance Structure:**

```
Executive Steering Committee
         ↓
   Project Sponsor
         ↓
   Project Manager
         ↓
  ┌──────┴──────┐
Core Team    Extended Team
```

**Decision Authority:**
- **Sponsor:** Budget changes >10%, scope changes affecting objectives
- **Steering Committee:** Strategic direction, escalated risks/issues
- **Project Manager:** Day-to-day decisions, resource allocation, minor scope
- **Core Team:** Technical decisions, implementation details

**Change Control Process:**
1. Change request submitted via [Tool/Form]
2. PM assesses impact (scope, schedule, budget, risk)
3. Approval required from:
   - Minor changes (<5% budget/schedule): PM approval
   - Moderate changes (5-10%): Sponsor approval
   - Major changes (>10%): Steering Committee approval
4. Approved changes update baseline, reject changes documented
