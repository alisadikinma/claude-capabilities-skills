# Hook Formulas Library

**Purpose:** 50+ proven hook templates to grab attention in 3-5 seconds

## The Science of Hooks

**Why Hooks Matter:**
- 70% of viewers decide to watch in first 3 seconds
- Pattern interrupt triggers dopamine response
- Curiosity gap forces completion

**Hook Success Metrics:**
- <30% drop-off in first 5 seconds = Excellent
- 30-50% drop-off = Good
- >50% drop-off = Rewrite hook

## Hook Writing Framework

### Step 1: Identify Core Value
What's the ONE thing viewers will gain?

### Step 2: Choose Hook Type
Match hook category to content type

### Step 3: Add Specificity
Replace generic with specific:
- "this trick" ‚Üí "this 3-line Python decorator"
- "faster" ‚Üí "10x faster (500ms to 50ms)"
- "better" ‚Üí "99.99% uptime vs 95%"

### Step 4: Test Against Criteria
- [ ] Under 5 seconds to say?
- [ ] Creates curiosity gap?
- [ ] Relevant to target audience?
- [ ] Specific and concrete?
- [ ] Emotionally engaging?

## Power Words for Hooks

**Urgency:** Now, Today, Immediately, Stop, Never, Always  
**Curiosity:** Secret, Hidden, Nobody, What, How, Why  
**Authority:** Expert, Pro, Proven, System, Method  
**Transformation:** From X to Y, Changed, Went, Become  
**Quantification:** 3-second, 10x, 80%, $1M, 500ms

**Emotion Triggers:**
- Fear: "Stop losing", "Before you waste"
- Greed: "Get more", "Increase", "Boost"
- Pride: "Level up", "Master", "Pro"
- Curiosity: "What if", "Imagine", "Discover"

## A/B Testing Hooks

**Test Method:**
1. Create 3 variants of same video with different hooks
2. Post at same time across 3 days
3. Compare retention at 5s mark
4. Winner = lowest drop-off rate

**Example Test:**
```
Version A: "You're using Docker wrong"
Version B: "This Docker mistake costs 2 hours daily"
Version C: "Docker deployment in 30 seconds - watch"

Results: Version B wins (72% retention vs 58%, 61%)
```
