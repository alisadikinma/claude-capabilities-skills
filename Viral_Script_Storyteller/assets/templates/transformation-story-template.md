### [0-5s] HOOK (Rock Bottom State)
**Visual:** Before photo/video OR metrics showing failure


## VISUAL STORYTELLING

### The 3-Act Visual Structure

**Act 1: Before State (0-15s)**
```
- Desaturated colors (dark, gloomy)
- Shaky cam or handheld feel
- Cluttered backgrounds
- Frustrated expressions
- Metrics in RED
```

**Act 2: Transformation (15-50s)**
```
- Colors gradually saturate
- Stabilized camera work
- Cleaner backgrounds
- Determined expressions
- Metrics transitioning (yellow â†’ green)
```

**Act 3: After State (50-60s)**
```
- Vibrant, saturated colors
- Smooth, professional camera
- Minimal, clean backgrounds
- Confident expressions
- Metrics in GREEN
```

**Color Psychology:**
- Red = failure, urgency
- Yellow = transition, caution
- Green = success, growth
- Blue = trust, stability

## PRODUCTION GUIDE

### Filming Checklist

**Before State Footage:**
- [ ] Old screenshots/photos gathered
- [ ] Failure metrics documented
- [ ] Authentic struggle evidence (emails, code, errors)
- [ ] Permission to show failures (if team/company involved)

**Transformation Process:**
- [ ] Time-lapse of work (code, design, testing)
- [ ] Key learning moment captured (reading, talking, testing)
- [ ] Obstacles documented (failures along the way)
- [ ] Progress tracked (version updates, metrics improving)

**After State Proof:**
- [ ] Current metrics screenshot
- [ ] User testimonials (if applicable)
- [ ] System running well (demo)
- [ ] Personal confidence on camera

**Technical Setup:**
- [ ] Consistent lighting across all timeline sections
- [ ] Audio quality checked (clear voice, no echo)
- [ ] Screen recordings high-res (1080p+)
- [ ] B-roll matches timeframe (don't fake timestamps)
